# Bookstore

Monorepo for the Bookstore project.

## Structure

```
bookstore/
├── app/        # Bookstore application
└── tests/      # Playwright end-to-end tests
```

## Getting started

Install all dependencies from the root:

```bash
npm install
```

### Run the app

```bash
npm run dev
```

### Run tests

```bash
npm run test
```
